Detailed Description

Dibuixa múltiples ObjecteJoc a una finestra.

Author
Roger Bassons Renart

La finestra(f) s'ha de crear amb crearFinestra(). f té una mida, un color de fons i un títol determinat.

Es pot afegir un KeyListener a f

Es poden afegir ObjecteJoc a DibuixadorAsteroides. Aquests ObjecteJoc es dibuixaran a f

La finestra es pot tornar a crear tantes vegades com es desitgi, però només s'utilitza la última que s'ha creat. La finestra es pot tancar en qualsevol moment.
