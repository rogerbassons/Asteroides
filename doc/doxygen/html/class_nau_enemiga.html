Detailed Description

Nau que té la capacitat d'atacar a una Nau i evitar Meteorit.

Comportament:

La NauEnemiga, per defecte, està parada i apunta i dispara a una Nau(n).

  • Si n es troba lluny de la NauEnemiga: la NauEnemiga s'apropa a n
  • Si n es troba aprop d'algun Meteorit(m): es desplaça en direcció contraria a m

lluny >= llargada de NauEnemiga * 2

aprop <= llargada de NauEnemiga * 10
